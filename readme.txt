== Description ==
Accept Mobile Money payments via the PawaPay Payment Page in WooCommerce. Supports multi-country, multi-currency, and automatic currency conversion using ExchangeRate API.

== Installation ==
1. Upload the plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Settings > Payments and configure PawaPay.
